# 🎮 GLOW BATTLE - 遊戲規則 (Game Rules)

## 📊 基本公式
```
半徑 (Radius) = 10 + √(分數) × 5
```

---

## 🍽️ 吞噬規則 (Eating Rules)

| 捕食者 | 獵物 | 條件 | 獎勵 |
|--------|------|------|------|
| 玩家 | 玩家 | 分數 > 對方分數 + 10%重疊 | 對方 **60%** 分數 |
| 玩家 | Bot | 分數 > Bot分數 + 碰撞 | Bot **40%** 分數 |
| 玩家 | 病毒 | 分數 > 病毒分數 + 碰撞 | **炸裂** 成 6 片 |
| 玩家 | 食物 | 碰撞 | **+1** 分數 |
| 玩家 | Jackpot | 碰撞 | **+50** 分數 |

---

## 🦠 病毒規則 (Virus Rules)

| 項目 | 數值 |
|------|------|
| 初始分數 | 100-135 (隨機) |
| 子彈餵養 | **+5** 分數/發 |
| 分裂門檻 | **135** 分數 |
| 噴發數量 | **3** 個 (-15°, 0°, +15°) |
| 噴發病毒分數 | 100-115 (隨機) |

---

## 🔘 細胞規則 (Cell Rules)

| 項目 | 數值 |
|------|------|
| 發射子彈消耗 (W鍵) | **5** 分數 |
| 發射最低分數 | **40** 分數 |
| 分裂最低分數 (Space) | **35** 分數 |
| 融合冷卻 | **10** 秒 |
| 融合重疊門檻 | **50%** |

---

## 📉 自然衰減規則 (Natural Decay)

| 分數區間 | 每秒衰減 | 毒圈外 (10倍) |
|----------|---------|--------------|
| 0-50     | 0       | 0            |
| 51-100   | 0.2     | 2.0/秒       |
| 101-150  | 0.3     | 3.0/秒       |
| 151-200  | 0.5     | 5.0/秒       |
| 201-300  | 0.8     | 8.0/秒       |
| 301+     | 1.0     | 10.0/秒      |

---

## 🎯 核心原則

> **只要你的分數比對方高，就能吃掉對方！**

無論是玩家、Bot 還是病毒，規則都一樣簡單。

---

## ⚠️ 注意事項

- Bot 會在地圖上隨機移動
- 病毒碰撞會讓你炸裂成多個碎片
- 碎片 10 秒後可以融合回來
- Jackpot (金色球) 稀有出現，價值 50 分
